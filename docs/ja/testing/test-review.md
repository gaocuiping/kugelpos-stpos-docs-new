# 総合テスト評審

> **バージョン:** 1.0 | **日付:** 2026-03-06 | **作成者:** QA チーム

## 1. エグゼクティブサマリー

本ドキュメントは、Kugelpos POS バックエンド全マイクロサービスのテストカバレッジと品質状況の総合評審レポートです。

| メトリクス | 状況 |
|-----------|------|
| カバー対象サービス | 8 / 8 |
| ユニットテストカバレッジ | 平均 ~65% |
| 統合テストカバレッジ | 平均 ~20% |
| シナリオテストカバレッジ | 平均 ~10% |
| 重大なギャップ | Terminal・Master-data ユニットテスト不足、Dapr pub/sub 統合テストなし |

---

## 2. テストピラミッド現状

```
          ┌───────────────┐
          │ シナリオ/E2E   │  ← 非常に少ない（Cart 一部のみ）
          │     ~10%      │
         ─┴───────────────┴─
        ┌─────────────────────┐
        │     統合テスト        │  ← 少ない（Account, Cart, Stock）
        │        ~20%          │
       ─┴─────────────────────┴─
      ┌─────────────────────────────┐
      │      ユニットテスト           │  ← 分布に偏りあり
      │           ~65%               │
      └─────────────────────────────┘
```

**課題:** ピラミッドが歪んでいる — 統合テストとシナリオテストが大幅に不足している。
